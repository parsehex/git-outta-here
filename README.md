# git-outta-here

![Screenshot of the game Git Outta Here](./goh-screenshot-3.png)

Have you ever wanted to **_play_** a GitHub profile? That's what this is -- at least in the form of an incremental game.

You can [play the game here](https://parsehex.github.io/git-outta-here/) - it automatically re-builds when I push to the `main` branch.

---

This was forked from [top-down-action-rpg-template](https://github.com/noidexe/top-down-action-rpg-template) by [noidexe](https://github.com/noidexe).
